## Description longue pour GitHub (README résumé — EN)

> **Cave & Vigne** is a full-stack personal wine cellar and spirits management app. Features a multi-provider AI sommelier (Claude, ChatGPT, Mistral, OpenWebUI/Ollama) for food/wine pairings and label scanning, a tasting journal, wishlist, CSV import/export, bulk operations, keep_until notifications, interactive D3.js vineyard maps (France + World), Chart.js dashboard, 2FA TOTP, password reset, public catalog mode, Redis caching, PWA, and a native Android WebView app.
>
> **Stack:** React 19 • Vite 6 • Bootstrap 5 • Node.js 24 • Express • MariaDB 12 • Redis • Nginx • Cloudflare
