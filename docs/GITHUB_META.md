## Description longue pour GitHub (README résumé — EN)

> **Cave & Vigne** is a full-stack personal wine cellar and spirits management app. Powered by Claude AI (Anthropic) for intelligent food/wine pairing suggestions and label scanning via computer vision. Features interactive D3.js vineyard maps (France + World), full CRUD for wines and spirits, JWT authentication, Redis caching, and a native Android WebView app.
>
> **Stack:** React 18 • Bootstrap 5 • Node.js • Express • MariaDB • Redis • Nginx • Cloudflare
